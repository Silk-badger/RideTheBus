# RideTheBus
Code Space/Repository for my final Ride The Bus project.

Hello, my name is Jack Towler. I'm starting this project officially on 3/2/26. I have tons of excitement and passion for this project.
I'm admist the creation, or recreation if you will, of a card game called "Ride The Bus". 
In the original drinking game, you guess cards per round and drink as the reward (or the dealer does.), however in this code-
You'll be wagering for a monetary increase following a luck based parameter game. 

How the game works is as follows:
at the beginning round, you decide the amount you'd like to wager throughout the game.
on the first round, you guess the color that will be pulled from a random card for a 2* reward.
on the second round, you guess if the next card will be higher or lower than the previous for a 3* reward.
on the third round, you guess if the next card will be inside or outside of the two previous cards for a 4* reward
and on the final round, you guess the suit of the last card for a *10 mega jackpot.

NOTE: This game is time based in real life, but wont be for this project.
(Inspiration comes from the in game version found in Schedule 1 (in the optional casino spot.))
